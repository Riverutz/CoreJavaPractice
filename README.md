# CoreJavaPractice

This project contains my practice and exercises while learning Core Java.

## Topics Covered
- Control Flow (if/else, switch, loops)
- Java Collections (List, Map)
- Objects, variables, and methods
- Access modifiers
- Exception handling
- OOP principles (Encapsulation, Inheritance, Polymorphism)
- Static keyword
- Value vs Reference types
- Lombok

## Purpose
The goal of this repository is to strengthen my understanding of Java fundamentals and build a solid foundation before moving to more advanced projects.

This project is continuously updated.
